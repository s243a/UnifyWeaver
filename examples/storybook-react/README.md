# UnifyWeaver Storybook Example

This directory contains a Storybook setup for documenting and testing the React components generated by UnifyWeaver.

## Quick Start

```bash
# Install dependencies
npm install

# Start Storybook
npm run storybook
```

Then open http://localhost:6006 in your browser.

## Structure

```
storybook-react/
├── .storybook/          # Storybook configuration
│   ├── main.ts          # Main config (addons, framework)
│   └── preview.ts       # Preview config (decorators, parameters)
├── src/
│   ├── components/      # Generated React components
│   │   └── ThemeToggle.tsx
│   ├── stories/         # Storybook stories
│   │   ├── Introduction.mdx
│   │   ├── ThemeToggle.stories.tsx
│   │   └── Animations.stories.tsx
│   └── styles/
│       └── globals.css  # Generated CSS variables
└── package.json
```

## Generating Components

Components can be regenerated from UnifyWeaver Prolog specifications:

```bash
# Generate components from UI primitives
swipl generate.pl
```

This runs the Prolog generator which creates React components in `src/components/generated/`:

- `Button.tsx` - Button variants (primary, secondary, danger, ghost)
- `TextInput.tsx` - Text input with labels and validation

The generator uses the UI primitives from `src/unifyweaver/ui/` to produce type-safe React components.

## Adding New Stories

1. Generate component code using UnifyWeaver
2. Save to `src/components/`
3. Create a story file in `src/stories/`

Example story structure:

```tsx
import type { Meta, StoryObj } from "@storybook/react";
import { MyComponent } from "../components/MyComponent";

const meta: Meta<typeof MyComponent> = {
  title: "Category/MyComponent",
  component: MyComponent,
  tags: ["autodocs"],
};

export default meta;
type Story = StoryObj<typeof MyComponent>;

export const Default: Story = {
  args: {
    // default props
  },
};
```

## Building for Production

```bash
npm run build-storybook
```

This creates a static Storybook site in `storybook-static/` that can be deployed to any static hosting service.
