# UnifyWeaver Visualization Components

Welcome to the Storybook for UnifyWeaver generated visualization components. This documentation showcases the React components generated by the UnifyWeaver Prolog-to-code generation system.

## Overview

UnifyWeaver generates production-ready visualization components from declarative Prolog specifications. The generated components include:

### Themes & Styling
- **Theme System** - Centralized theme definitions with CSS custom properties
- **Animation Presets** - Entry, exit, and attention animations
- **Template Library** - Pre-built dashboard and report templates

### Performance
- **Lazy Loading** - Pagination, infinite scroll, chunked loading
- **Virtual Scrolling** - Efficient rendering for large lists/tables/grids
- **WebWorkers** - Background data processing

### Charts & Visualizations
- Line, Bar, Pie, Area charts
- Heatmaps and Treemaps
- 3D plots and surfaces
- Interactive features (zoom, pan, tooltips)

## Getting Started

1. Generate components from Prolog specifications:

```prolog
% Generate a themed line chart
?- generate_visualization(my_chart, line_chart, [
    theme(dark),
    responsive(true),
    animation(fade_in)
], Code).
```

2. Copy generated code to your React project
3. Import and use components
