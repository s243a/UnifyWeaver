# react_cli

Generated by UnifyWeaver Project Scaffold Generator.

## Generation

The application files can be regenerated from Prolog specifications:

```bash
# Generate into generated/ directory
./generate.pl
```

This uses `react_cli_module.pl` to generate `generated/src/App.tsx` using the UnifyWeaver declarative UI component generator.

### Directory Layout

- `prototype/` — Hand-crafted reference implementation
- `generated/` — Output from `generate.pl` (do not edit by hand)
- `src/` — Active working copy (may be either prototype or generated)

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
