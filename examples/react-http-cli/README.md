# React HTTP CLI

A React frontend for the UnifyWeaver HTTP CLI server, comparable to the Vue version.

## Features

- **Browse** - Navigate directories, select files, download
- **Upload** - Upload files to server
- **Cat** - View file contents
- **Shell** - Interactive WebSocket shell (requires shell role)

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Then open http://localhost:5174

## Configuration

Edit `src/App.tsx` and update `API_BASE` to point to your HTTP CLI server:

```typescript
const API_BASE = 'https://localhost:3000'
```

## Usage

1. Start the HTTP CLI server first:
   ```bash
   cd ../http-cli-server/generated
   npm start
   ```

2. Start the React app:
   ```bash
   npm run dev
   ```

3. Login with default credentials: `shell@local` / `shell`

## Tabs

| Tab | Description | Required Role |
|-----|-------------|---------------|
| Browse | File browser with navigation | user |
| Upload | Upload files to working directory | admin, shell |
| Cat | View file contents | user |
| Shell | Interactive terminal | shell |

## Comparison to Vue Version

This React app provides the same core functionality as the Vue version generated by UnifyWeaver, demonstrating multi-target UI generation capability.
