# AI Skills Documentation

This folder contains documentation for AI assistants on how to perform specialized tasks within the UnifyWeaver project.

## Purpose

These documents teach LLMs how to:
1. Act as orchestrators for multi-model workflows
2. Execute specialized testing and validation tasks
3. Coordinate between different AI models and tools
4. Follow project-specific patterns and conventions

## Available Skills

| Skill | Description | Complexity |
|-------|-------------|------------|
| [Test Orchestrator](./test-orchestrator.md) | Coordinate playbook testing across multiple LLMs | Medium |
| [Bookmark Filing Agent](./bookmark-filing-agent.md) | File bookmarks into Pearltrees using semantic search + LLM | Low |

## Skill Document Structure

Each skill document follows this structure:

1. **Overview** - What the skill does and when to use it
2. **Prerequisites** - Required tools, access, and context
3. **Step-by-Step Instructions** - Detailed procedures
4. **Model-Specific Commands** - How to invoke different models
5. **Recording Results** - How to document outcomes
6. **Troubleshooting** - Common issues and solutions

## For AI Assistants

When asked to perform a task covered by these skills:
1. Read the relevant skill document
2. Follow the procedures exactly
3. Record results in the specified format
4. Update documentation as needed
